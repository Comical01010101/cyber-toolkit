# Cyber Mini Defense Toolkit

## Overview
This project was developed as part of the GET IT Intro to Cybersecurity program.
The Cyber Mini-Defense Toolkit demonstrates foundational cybersecurity skills 
using Python, including password security, encryption, log analysis, and threat detection.

---

## Features
- Password Strength Checker & Hashing
- Caesar Cipher Encryption/Decryption Tool
- Log Analyzer for Suspicious Activity
- Basic Phishing Detection Logic

---

## Technologies Used
- Python (Jupyter Notebook)
- hashlib
- pandas
- logging
- scikit-learn

---

## How to Run the Project
1. Open Jupyter Notebook
2. Navigate to the project folder
3. Open each notebook file:
   - password_checker.ipynb
   - cipher_tool.ipynb
   - log_analyzer.ipynb
4. Run each notebook cell using Shift + Enter

## Cybersecurity Concepts Demonstrated
- CIA Triad (Confidentiality, Integrity, Availability)
- Password Security & Hashing
- Encryption & Decryption
